## Description

A React app built to practice React Router. Users can list their bikes, view bike cards, and navigate through pages for hosting, browsing, and renting.

## Features

- Users can host/list a bike
- Browse bike cards with details
- Basic renting flow
- Multiple pages handled with React Router
- Simple, clean UI built for learning routing concepts

## Tech Stack

React, React Router, JavaScript, CSS.

## Setup

```bash
git clone https://github.com/jxkz3/ReactProjects.git
cd bike-life
npm install
npm run dev
```
