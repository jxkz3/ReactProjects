# Word Stack Game

A simple React project where the player guesses letters to build the correct word. Duplicate guesses are ignored, and the stack grows as new letters are added.

## Features

- Tracks guessed letters
- Prevents duplicates
- Clean and minimal UI
- Built with React (Vite)

## Run the Project

`npm install`
`npm run dev `

## Tech Stack

React, JavaScript, CSS

## Purpose

A small practice project focused on React state handling and component structure.
