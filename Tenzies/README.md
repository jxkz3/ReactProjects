# Tenzies Game

A simple React version of the classic Tenzies game. Roll the dice until all numbers match. Hold dice you want to keep and reroll the rest.

## How to Play

- Click a die to hold it
- Press "Roll" to reroll the unheld dice
- Win when all dice show the same number

## Run the Project

`npm install`
`npm run dev`

## Tech Stack

React, JavaScript, CSS

## Purpose

A small practice project to improve state management, component structure, and game logic in React.
